## General info
Simple application to save money and manage home capital.

## Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# run linting
$ npm run lint
$ npm run lint:fix

# generate static project
$ npm run generate
```
